# Setting Up Metadata for SEO and Social Sharing

Proper metadata configuration is crucial for making your documentation site discoverable on search engines and look great when shared on social media. This guide will walk you through setting up metadata in your **Next.js** project.

## Why Metadata Matters

- Improves search engine rankings (SEO)
- Ensures rich previews when shared on Twitter, LinkedIn, and Facebook
- Helps define authorship and branding

## Configuring Metadata in Next.js

Next.js provides built-in support for metadata configuration via the [`metadata` API](https://nextjs.org/docs/app/api-reference/functions/generate-metadata).

### Step 1: Create or Change Existing Metadata Configuration File

Inside your project, create a new file at:

### Step 2: Verify Your Metadata

Once you've set up the metadata, verify it using:

- [Google Rich Results Test](https://search.google.com/test/rich-results)
- [Twitter Card Validator](https://cards-dev.twitter.com/validator)
- [Facebook Sharing Debugger](https://developers.facebook.com/tools/debug/)
