# Context7 Library Registry

Este arquivo mantém o registro de todas as bibliotecas utilizadas via Context7 no projeto. **Sempre verifique este arquivo antes de buscar documentação de bibliotecas via Context7.**

## Processo de Uso

1. **Antes de buscar**: Verifique se a biblioteca já está listada abaixo
2. **Se não estiver**: Busque o ID via Context7, use a biblioteca, e **adicione imediatamente** nesta lista
3. **Manutenção**: Atualize este arquivo a cada sprint durante code review

## Formato de Entrada

Ao adicionar uma nova biblioteca, use o formato:

```markdown
| `library-id` | Nome da Biblioteca | Breve descrição | YYYY-MM-DD | Notas adicionais (opcional) |
```
